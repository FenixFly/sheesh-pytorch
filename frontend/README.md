# frontend часть проекта SHEESH
## Зависимости
nodejs v14<br>
webpack v5

## Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
quasar dev
```


### сборка в режиме SPA
```bash
quasar build
```

### сборка exe в режиме Electron
```bash
quasar build --mode electron
```
